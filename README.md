# DevOps Course Progress

Suivi de ma progression dans la formation **"Decoding DevOps – From Basics to Advanced Projects with AI"** (Imran Teli, Udemy), documentée au fil de l'eau : setup, notes techniques, et projets pratiques.

## Pourquoi ce repo

Ce n'est pas un tutoriel recopié — c'est une trace de mon propre apprentissage : les choix faits, les erreurs rencontrées, et comment je les ai résolues. L'objectif est double : garder une référence pour moi-même, et montrer une méthode de travail plutôt qu'un simple certificat de fin de cours.

## Stack couverte par la formation

Linux & administration serveur, réseau, Vagrant, Bash/YAML/JSON, AWS (IAM, EC2, S3, RDS, VPC, Auto Scaling...), Git, Jenkins & GitHub Actions, Python, Terraform, Ansible, Docker & Kubernetes, Prometheus/Grafana, GitOps (ArgoCD).
